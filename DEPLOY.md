1. Browser open: **https://render.com**
2. **Get Started** → **Sign up with GitHub**
3. GitHub account `albertraj163` connect pannunga

## Step 2 — Deploy (one click)

1. Open: **https://dashboard.render.com/select-repo?type=blueprint**
2. Repo select pannunga: **`car-price-prediction`**
3. **Apply** click pannunga
4. 2–3 minutes wait pannunga — deploy complete aagum
